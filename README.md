# GMAura — своя коллекция обоев

GMAura умеет показывать обои не только из встроенной коллекции, но и из **вашей собственной папки на Яндекс Диске**. Вы складываете в неё видео и картинки, делаете папку публичной и добавляете ссылку в приложении — обои появляютсяавтоматически.

Эта инструкция объясняет, как устроена такая папка: структура, форматы, разрешение и правила именования файлов.

---

## 1. Как добавить свою папку в приложение

1. Создайте папку на [Яндекс.Диске](https://disk.yandex.ru).
2. Включите для неё **общий доступ** (публичную ссылку): правый клик по папке → «Поделиться» → скопировать ссылку. Ссылка будет вида `https://disk.yandex.ru/d/XXXXXXXXXXXX`.
3. В приложении GMAura: экран источников → добавить источник → вставьте название и эту ссылку.

> Подойдёт ссылка и `https://disk.yandex.ru/d/...`, и `https://disk.yandex.com/d/...`.

---

## 2. Структура папки

Внутри вашей публичной папки приложение ищет три подпапки:

```
Моя коллекция/
├── Videos/    — видеообои (.mp4)
├── Images/    — картинки-обои (.png / .jpg / .jpeg)
└── thumbs/    — превью для списка (необязательно)
```

- Любую подпапку можно **не создавать** — если нет `Videos/`, просто не будет видео.
- Имена подпапок важны: `Videos`, `Images`, `thumbs` (именно так).

---

## 3. Форматы и разрешение

| Подпапка   | Что кладём        | Форматы              |
|------------|-------------------|----------------------|
| `Videos/`  | видео             | `.mp4`               |
| `Images/`  | картинки          | `.png`, `.jpg`, `.jpeg` |
| `thumbs/`  | превью (миниатюры) | `.png`, `.jpg`, `.jpeg` |

**Рекомендуемое разрешение — 1920×1080 на каждый экран.** У головного устройства два экрана; для парных обоев (см. ниже) делайте два файла по 1920×1080.

---

## 4. Правила именования

Тип обоев определяется **именем файла**.

### Одиночные

Любое имя без специальных суффиксов. Такие обои можно поставить на любой экран по отдельности.

```
Videos/aurora.mp4
Images/mountains.jpg
```

### Парные — на два экрана сразу

Два файла с одинаковым именем и суффиксами `_l` (левый экран) и `_r` (правый экран) объединяются в одни обои, которые ставятся сразу на оба экрана.

```
Videos/aurora_l.mp4    ← левый экран
Videos/aurora_r.mp4    ← правый экран
```

Если пары нет (есть только `_l` или только `_r`) — файл считается одиночным.

### Динамические — смена по времени суток

Несколько файлов одной обои с суффиксом `_d_ЧЧММ` — это «слоты», между которыми приложение само переключается по часам (24-часовой формат). `ЧЧ` = 00–23, `ММ` = 00–59.

```
Images/catalina_d_0600.jpg    ← с 06:00 (утро)
Images/catalina_d_1200.jpg    ← с 12:00 (день)
Images/catalina_d_2000.jpg    ← с 20:00 (вечер)
```

- Слот действует с указанного времени до начала следующего; после последнего цикл замыкается на первый.
- Слоты можно складывать и в `Videos/`, и в `Images/`, и **смешивать** видео и картинки в одном наборе.

Для обоев день/ночь, рекомендуется использовать слоты: `0600` и `1800`.

```
Images/catalina_d_0600.jpg    ← 06:00 (день)
Images/catalina_d_1800.jpg    ← 18:00 (ночь)
```

Для остальных обоев слоты: `0000`, `0300`, `0600`, `0900`, `1200`, `1500`, `1800`, `2100`.

### Динамические + парные

Комбинация: сторона `_l`/`_r` ставится **перед** меткой времени `_d_ЧЧММ`.

```
Images/catalina_l_d_1200.jpg
Images/catalina_r_d_1200.jpg
```

---

## 5. Превью (`thumbs/`)

Чтобы в списке обоев показывалась миниатюра, положите в `thumbs/` картинку с **тем же именем, что у файла обоев**:

```
Videos/aurora_l.mp4   →   thumbs/aurora_l.jpg
```

Можно сделать одно общее превью на пару/набор — по «базовому» имени:

```
thumbs/aurora.jpg       ← подойдёт для aurora_l / aurora_r
thumbs/catalina.jpg     ← подойдёт для всех слотов catalina_d_*
```

Превью необязательны, но очень желательны. Если их нет, приложение возьмёт сам файл обоев.

---

## 6. Как имя превращается в название в приложении

Подчёркивания и дефисы становятся пробелами, каждое слово — с заглавной буквы:

| Имя файла          | Название в приложении |
|--------------------|-----------------------|
| `aurora.mp4`       | Aurora                |
| `aurora_bay.mp4`   | Aurora Bay            |
| `night-city_l.jpg` | Night City            |

Суффиксы `_l`, `_r`, `_d_ЧЧММ` в названии не показываются.

---

## 7. Полный пример

```
Моя коллекция/
├── Videos/
│   ├── aurora_l.mp4              ┐ парные обои «Aurora»
│   └── aurora_r.mp4              ┘
├── Images/
│   ├── mountains.jpg            → одиночные «Mountains»
│   ├── catalina_d_0600.jpg     ┐
│   ├── catalina_d_1200.jpg     │ динамические «Catalina» (3 слота)
│   └── catalina_d_2000.jpg     ┘
└── thumbs/
    ├── aurora.jpg               → превью для пары Aurora
    ├── mountains.jpg            → превью для Mountains
    └── catalina.jpg            → превью для набора Catalina
```

В приложении это даст три обои:

- **Aurora** — парные видеообои (левый + правый экран);
- **Mountains** — одиночная картинка;
- **Catalina** — динамическая, меняется в 06:00, 12:00 и 20:00.
